# Personal One-Pager Website

## Overview
A premium, minimal one-page personal website for Anastasiia Petrashevskaia, designed to help her get hired into senior roles in supplier enablement, procurement transformation, compliance, and global process leadership.

## Project Architecture
- **Type**: Static website
- **Stack**: HTML, CSS, JavaScript
- **Server**: Python SimpleHTTPServer on port 5000

## File Structure
- `index.html` - Main HTML page with all content sections
- `styles.css` - Styling with "Quiet Luxury Minimal" theme
- `script.js` - Subtle animation effects
- `server.py` - Simple Python HTTP server for development
- `spec.md` - Original specification document

## Key Sections
1. Hero - Name, title, positioning statement, CTAs
2. Proof Highlights - 4 key impact metrics
3. Selected Experience - 6 career highlights with outcomes
4. Capabilities - Core competency chips
5. Education & Tools - Academic background and tools
6. Contact - Final CTA section

## Running the Project
The website runs via the "Website" workflow using `python server.py` on port 5000.
